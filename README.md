# 🌍 World Countries Analysis

Data analysis of 91 world countries using Python, Pandas & Matplotlib.

## 📊 What this project covers
- Loaded and cleaned a real world dataset
- Analysed population, area, literacy and GDP
- Created 5 visualizations including bar charts, pie chart and histogram

## 🔍 Key Findings
- China is the most populous country (1.3 billion)
- Russia is the largest country by area (17M sq km)
- Western Europe is the wealthiest region by GDP per capita
- 11 countries have a literacy rate below 50%

## 🛠 Tools Used
- Python, Pandas, Matplotlib, Jupyter Notebook
